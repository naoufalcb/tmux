# tmux Config

## Requirements

Install tmux if not already installed:

**Ubuntu/Debian**
```bash
sudo apt install tmux
```

**Arch Linux**
```bash
sudo pacman -S tmux
```

**macOS**
```bash
brew install tmux
```

## Installation

```bash
git clone https://github.com/naoufalcb/tmux.git ~/.config/tmux
tmux source ~/.config/tmux/tmux.conf
```

> If tmux is not running yet, just start a new session with `tmux` — the config loads automatically.
